# 🤖 Fluxia AI

## ✏️ Project Description

💡 This project is a mock of an AI Business Automation website where users can view differents company features.

📑 The website includes six sections: The Problem, Solutions, Use Cases, How It Works, Pricing, Contact

📱 It is fully responsive, accessible, and optimized for cross-platform use.

🚀 The website is deployed on Vercel.

🌐 You can visit the website at the following link ➡️ https://fluxia-ai.vercel.app/


## 🛠️ Technology Stack

- Angular 21
- TypeScript
- JavaScript
- HTML5
- CSS3 
- Tailwind CSS
- OpenCode
- Figma Make
- Lucide Icons
- Prettier
- Vercel


## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
ng serve
```

### Type-Check, Compile and Minify for Production

```sh
ng build
```
